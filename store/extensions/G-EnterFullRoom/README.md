# G-EnterFullRoom

With this extension you can enter a full room **automatically** when someone leave the room.

## How to use?
1.  Enter the **full** room you want to get in.
2.  Click on **"Activate"**.
3.  **Now just wait!**
You will get automatically in, when someone leave the room.
